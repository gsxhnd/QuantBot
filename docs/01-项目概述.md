# 项目概述

## 1. 项目简介

本项目旨在构建一个完全本地化的中长期量化交易系统,具备 7x24 小时盯盘能力,可进行新闻分析并提供买卖建议。系统设计满足个人投资者对自动化交易的需求,确保数据安全和隐私保护。

## 2. 目标用户

- 个人投资者(自行使用)
- 注重数据隐私和本地化部署
- 需要自动化交易决策支持

## 3. 硬件平台

**Jetson Orin Nano Super**

- CPU: 6 核 ARM Cortex-A78AE
- 内存: 8GB LPDDR5
- GPU: 512-core NVIDIA Ampere GPU
- 适合边缘 AI 推理和本地部署

## 4. 核心特性

### 4.1 实时监控

- 7x24 小时市场监控
- 多市场支持(A股、美股)
- 实时行情数据处理

### 4.2 智能分析

- 基于深度学习的价格预测
- 新闻情绪分析
- 多因子量化策略

### 4.3 风险管理

- VaR 风险度量
- 动态头寸管理
- 实时回撤监控
- 自动止损止盈

### 4.4 自动化执行

- 智能订单拆分
- 交易成本优化
- 执行容错机制
- 自动对账

## 5. 技术亮点

- **完全本地化**: 所有数据和计算在本地完成,无隐私泄露风险
- **高性能推理**: 基于 ONNX Runtime 的 GPU 加速推理
- **混合架构**: Python 训练 + Rust 推理,兼顾灵活性和性能
- **嵌入式数据库**: DuckDB 零配置,列式存储优化查询性能
- **模块化设计**: 易于扩展和维护

## 6. 应用场景

- 个人量化交易策略研发
- 多策略组合管理
- 历史数据回测验证
- 实盘自动化交易
- 投资组合风险监控
