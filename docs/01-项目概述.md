# 项目概述

## 1. 项目简介

完全本地化的中长期量化交易系统，具备 7x24 小时盯盘能力，可进行新闻分析并提供买卖建议。

**核心价值**：
- 完全本地运行，数据隐私安全
- AI 驱动的智能决策
- 自动化风险控制
- 个人投资者可用

## 2. 硬件平台

**Jetson Orin Nano Super**
- CPU: 6 核 ARM Cortex-A78AE
- 内存: 8GB LPDDR5
- GPU: 512-core NVIDIA Ampere GPU

## 3. 核心功能

### 3.1 市场监控
- 7x24 小时实时盯盘
- 多市场支持（A股、美股）
- 异常波动预警

### 3.2 智能分析
- 深度学习价格预测（GRU 模型）
- 新闻情绪分析
- 多因子量化策略

### 3.3 风险管理
- VaR 风险度量
- Kelly 准则头寸管理
- 实时回撤监控
- 自动止损止盈

### 3.4 交易执行
- 智能订单拆分（TWAP/VWAP）
- 交易成本优化
- 执行容错和对账

### 3.5 系统监控
- 模型漂移检测
- 自动重训练
- 多级告警系统

## 4. 技术特点

- **Python + Rust 混合架构**：训练灵活，推理高效
- **ONNX 模型格式**：跨语言部署
- **DuckDB 嵌入式数据库**：零配置，列式存储
- **完全本地化**：无隐私泄露风险
