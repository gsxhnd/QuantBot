# 回测框架

## 1. 回测框架概述

### 1.1 回测目的

- 验证策略有效性
- 评估策略风险收益特征
- 优化策略参数
- 估算实盘表现

### 1.2 回测原则

**避免前瞻偏差**:

- 使用时间序列交叉验证
- 严格按时间顺序模拟
- 不使用未来数据

**真实性原则**:

- 包含完整交易成本
- 模拟真实滑点
- 考虑流动性约束
- 模拟市场冲击

## 2. 回测方法

### 2.1 时间序列交叉验证

**Walk-Forward 分析**:

```
训练窗口: 252天 (1年)
测试窗口: 63天 (3个月)
滚动步长: 21天 (1个月)

[训练1][测试1]
    [训练2][测试2]
        [训练3][测试3]
```

**优点**:

- 防止过拟合
- 评估策略稳定性
- 模拟真实场景

### 2.2 蒙特卡洛模拟

**模拟流程**:

1. 基于历史收益率拟合分布
2. 随机生成 N 条收益路径 (N=10000)
3. 计算每条路径的绩效指标
4. 统计分布特征

**应用**:

- 评估尾部风险
- 计算置信区间
- 压力测试

### 2.3 压力测试

**历史情景重放**:

- 2008年金融危机
- 2015年A股股灾
- 2020年疫情暴跌
- 评估极端情况表现

**假设情景测试**:

- 市场暴跌 20%
- 波动率翻倍
- 流动性枯竭
- 测试策略韧性

## 3. 回测指标

### 3.1 收益指标

| 指标 | 计算公式 | 说明 |
|------|---------|------|
| 总收益率 | (期末资产 - 期初资产) / 期初资产 | 累计收益 |
| 年化收益率 | (1 + 总收益率)^(252/交易日数) - 1 | 标准化收益 |
| 月均收益率 | 年化收益率 / 12 | 月度平均 |
| 超额收益 | 策略收益 - 基准收益 | 相对表现 |

### 3.2 风险指标

| 指标 | 计算公式 | 说明 |
|------|---------|------|
| 波动率 | std(日收益率) × √252 | 年化波动 |
| 最大回撤 | max(峰值 - 当前值) / 峰值 | 最大损失 |
| 下行波动率 | std(负收益) × √252 | 下行风险 |
| VaR(95%) | 收益分布 5% 分位数 | 风险价值 |
| CVaR(95%) | VaR 以下平均损失 | 期望损失 |

### 3.3 风险调整收益

| 指标 | 计算公式 | 目标值 |
|------|---------|--------|
| 夏普比率 | (年化收益 - 无风险利率) / 波动率 | > 1.0 |
| 索提诺比率 | (年化收益 - 无风险利率) / 下行波动率 | > 1.5 |
| 卡玛比率 | 年化收益 / 最大回撤 | > 0.5 |
| 信息比率 | 超额收益 / 跟踪误差 | > 0.5 |

### 3.4 交易指标

| 指标 | 说明 | 目标值 |
|------|------|--------|
| 总交易次数 | 买入+卖出次数 | - |
| 胜率 | 盈利交易 / 总交易 | > 50% |
| 盈亏比 | 平均盈利 / 平均亏损 | > 1.5 |
| 盈利因子 | 总盈利 / 总亏损 | > 1.5 |
| 最大连续亏损 | 连续亏损笔数 | < 5 |
| 平均持仓天数 | 总持仓天数 / 交易次数 | - |

### 3.5 统计指标

| 指标 | 说明 |
|------|------|
| 偏度 | 收益分布偏斜程度 |
| 峰度 | 收益分布尾部厚度 |
| 相关系数 | 与基准的相关性 |
| Beta | 系统性风险暴露 |

## 4. 成本模拟

### 4.1 佣金成本

```python
# A股
买入佣金 = max(成交金额 × 0.0003, 5)
卖出佣金 = max(成交金额 × 0.0003, 5)
印花税 = 成交金额 × 0.001  # 仅卖出
过户费 = 成交金额 × 0.00001

# 美股
佣金 = 股数 × 0.005
```

### 4.2 滑点成本

```python
# 基础滑点
基础滑点率 = 0.001  # 0.1%

# 波动率调整
波动率系数 = 当前波动率 / 历史平均波动率

# 时段调整
时段系数 = 1.5 if 开盘或收盘 else 1.0

# 最终滑点
滑点 = 成交金额 × 基础滑点率 × 波动率系数 × 时段系数
```

### 4.3 流动性成本

```python
# Market Impact
订单占比 = 订单量 / 日均成交量
流动性成本 = 成交金额 × 0.1 × (订单占比 ** 0.6)
```

### 4.4 总成本计算

```python
总成本 = 佣金 + 印花税 + 过户费 + 滑点 + 流动性成本
净收益 = 策略收益 - 总成本
```

## 5. 回测流程

### 5.1 数据准备

1. 加载历史行情数据
2. 计算技术指标
3. 数据清洗和对齐
4. 划分训练/测试集

### 5.2 策略初始化

1. 设置初始资金
2. 配置策略参数
3. 初始化持仓
4. 设置交易规则

### 5.3 回测执行

```python
for 每个交易日:
    # 1. 更新市场数据
    更新行情数据
    
    # 2. 生成交易信号
    信号 = 策略.生成信号(数据)
    
    # 3. 风险检查
    if 风险管理.检查(信号):
        # 4. 生成订单
        订单 = 生成订单(信号)
        
        # 5. 模拟执行
        成交 = 模拟执行(订单, 成本模型)
        
        # 6. 更新持仓
        更新持仓(成交)
    
    # 7. 计算绩效
    计算当日收益和风险指标
    
    # 8. 记录日志
    记录交易和持仓
```

### 5.4 结果分析

1. 计算绩效指标
2. 生成收益曲线
3. 分析回撤
4. 评估交易质量
5. 生成回测报告

## 7. 参数优化

### 7.1 网格搜索

```python
# 定义参数空间
参数空间 = {
    '窗口期': [10, 20, 30, 60],
    '阈值': [0.01, 0.02, 0.03],
    '止损': [0.02, 0.05, 0.10]
}

# 遍历所有组合
for 参数组合 in 参数空间:
    回测结果 = 回测(参数组合)
    记录结果
    
# 选择最优参数
最优参数 = 夏普比率最高的组合
```

### 7.2 贝叶斯优化

- 更高效的参数搜索
- 减少计算量
- 适合连续参数空间

### 7.3 过拟合防范

**防范措施**:

- 使用样本外测试
- 限制参数搜索范围
- 增加正则化约束
- 交叉验证

**验证方法**:

- 训练集和测试集表现对比
- 不同时间段稳定性
- 参数敏感性分析

## 8. 回测注意事项

### 8.1 常见陷阱

- **前瞻偏差**: 使用未来数据
- **幸存者偏差**: 只用存活股票
- **数据窥探**: 过度优化参数
- **成本低估**: 忽略真实成本
- **流动性忽视**: 忽略大单冲击

### 8.2 最佳实践

- 使用真实完整数据
- 包含完整交易成本
- 考虑流动性约束
- 多时段验证
- 保持策略简单
- 记录完整日志
