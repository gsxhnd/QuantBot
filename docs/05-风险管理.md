# 风险管理

## 1. 风险管理框架

**三层控制**：账户级（总资金、总杠杆）→ 策略级（单策略预算）→ 持仓级（单股限制）

**核心理念**：事前控制、事中监控、事后分析

## 2. 核心风险指标

| 指标 | 定义 | 目标阈值 | 监控频率 |
|------|------|--------|--------|
| VaR (95%) | 95% 置信度最大亏损 | < 账户 2% | 每分钟 |
| 最大回撤 | 峰值到谷底最大跌幅 | < 20% | 每日 |
| 夏普比率 | 风险调整收益 | > 1.0 | 每日 |
| 卡玛比率 | 年收益/最大回撤 | > 0.5 | 每日 |
| 头寸集中度 | 单股票敞口 | < 5% | 每笔交易 |
| 总杠杆率 | 全部头寸/账户资产 | < 2.0 | 每笔交易 |
| 实时回撤 | 当前值与当日高点差 | < 日均 150% | 每分钟 |

**VaR 计算方法**：历史模拟法、参数法（正态分布）、蒙特卡洛模拟

## 3. 头寸管理

### 3.1 Kelly 准则

**公式**：f* = (p × b - q) / b
- p = 胜率，q = 败率，b = 盈亏比
- 实际使用 Kelly 的 1/4 或 1/2（保守）

### 3.2 头寸约束

| 约束类型 | 限制 |
|---------|------|
| 单股票 | ≤ 账户 5%，高波动股票降低 |
| 单行业 | ≤ 账户 20% |
| 总杠杆 | ≤ 2.0，高波动期降低 |

## 4. 止损止盈

**止损策略**：
- 固定止损：单笔最大亏损 2%
- 移动止损：盈利后上移保护利润
- 时间止损：持仓超 N 天未盈利平仓

**止盈策略**：
- 目标止盈：达到预期收益率（5%、10%）
- 分批止盈：盈利后分批减仓

## 5. 风险预警和应急响应

| 级别 | 触发条件 | 响应动作 |
|------|---------|---------|
| **WARNING** | 夏普比 < 1.0、单日回撤 > 3%、连续 3 天负收益 | 邮件通知、记录日志 |
| **ERROR** | 实时回撤 > 日均 150%、头寸 > 10%、杠杆 > 3.0 | 即时推送、仪表板高亮、模拟盘减仓 50% |
| **CRITICAL** | VaR 突破、单日亏损 > 5%、系统异常 | 紧急推送、自动平仓、停止交易 |

**应急预案**：
- 市场极端波动：停止开仓、评估风险、对冲减仓
- 系统故障：切换备用、手动监控、必要时平仓
- 数据异常：停止交易、切换数据源、验证质量

## 6. 压力测试

- 历史情景：2008 金融危机、2020 疫情、2015 A 股股灾
- 假设情景：市场暴跌 20%、持仓跌停、流动性枯竭
- 蒙特卡洛：模拟 10000 条路径，评估尾部风险
