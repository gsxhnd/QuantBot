# 项目概述

## 1. 项目简介

完全本地化的中长期量化分析系统，具备7x24小时市场监控能力，提供基于AI的投资建议。

**核心价值**：

- 完全本地运行，数据隐私安全
- AI驱动的智能分析
- 自动化风险预警
- 个人投资者可用

**系统定位**：

QuantBot 是一个**量化分析系统**，而非自动交易系统。系统提供投资建议，用户负责最终决策和交易执行。

## 2. 硬件平台

**Jetson Orin Nano Super**

- CPU: 6核 ARM Cortex-A78AE
- 内存: 8GB LPDDR5
- GPU: 512-core NVIDIA Ampere GPU

## 3. 核心功能

### 3.1 市场监控

- 7x24小时实时盯盘
- 多市场支持（A股、美股）
- 异常波动预警
- 价格监控告警（止损/止盈触发）

### 3.2 智能分析

- 深度学习价格预测（GRU模型）
- 多因子量化策略
- 投资建议生成

### 3.3 风险管理

- VaR风险度量
- Kelly准则仓位建议
- 实时回撤监控
- 止损止盈价格建议与告警

### 3.4 建议输出

- 买入/卖出信号
- 置信度评估
- 建议止损价、目标价
- 仓位建议

### 3.5 系统监控

- 模型漂移检测
- 自动重训练
- 多级告警系统

### 3.6 外部集成

- 新闻抓取与情绪分析
- 信号推送与用户通知

### 3.7 财报分析

- 财务数据获取与存储（三大财务报表、关键指标）
- 财报质量评估（盈利质量、资产负债质量、现金流质量）
- 财务异常检测（Benford定律、波动异常、行业对比）
- 财报因子集成到投资建议

### 3.8 情绪分析增强

- 社交媒体情绪监测（Twitter/X、Reddit、雪球）
- 多源情绪聚合（新闻+社交媒体加权融合）
- 情绪因子深度融合到AI模型预测
- 情绪趋势分析与异常监控

## 4. 技术特点

- **Python + Rust 混合架构**：训练灵活，推理高效
- **ONNX 模型格式**：跨语言部署
- **DuckDB 嵌入式数据库**：零配置，列式存储
- **完全本地化**：无隐私泄露风险
- **建议驱动**：AI辅助决策，用户掌控交易

## 5. 系统角色

| 角色 | 职责 |
|------|------|
| **QuantBot** | 数据分析、信号生成、风险评估、告警触发 |
| **用户** | 接收建议、评估风险、手动交易 |
