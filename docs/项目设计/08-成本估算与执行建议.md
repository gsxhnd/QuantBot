# 成本估算与执行建议

## 1. 成本估算模型

**总成本** = 佣金 + 滑点 + 流动性成本

### 1.1 佣金成本

| 市场 | 费用明细 |
|------|---------|
| **A股** | 买卖佣金0.03%（最低5元）+ 印花税0.1%（卖出）+ 过户费0.001% |
| **美股** | 佣金$0.005/股 + SEC费用$0.0000278/美元 |

### 1.2 滑点估算

**公式**：滑点 = 基础滑点 × 波动率系数 × 时段系数

- 基础滑点：0.1% - 0.3%
- 波动率系数：当前波动率 / 历史平均
- 时段系数：开盘/收盘1.5，盘中1.0

### 1.3 流动性成本

**Market Impact**：流动性成本 = α × (订单量 / 日均成交量)^β

- 高流动性：α=0.1, β=0.5
- 中流动性：α=0.3, β=0.6
- 低流动性：α=0.5, β=0.7

## 3. 执行时机建议

### 3.1 最佳执行时段

| 时段 | 建议 | 原因 |
|------|------|------|
| 开盘前 | 观望 | 流动性差，波动大 |
| 开盘后30分钟 | 适合大单 | 流动性改善 |
| 盘中 | 适合小单 | 流动性好，波动小 |
| 收盘前30分钟 | 谨慎 | 波动增大 |

### 3.2 执行策略建议

| 订单类型 | 建议策略 | 说明 |
|---------|---------|------|
| 小单（< 日均1%） | 直接市价 | 成本低，执行快 |
| 中单（1%-10%） | TWAP分批 | 分3-5批，每批间隔5-10分钟 |
| 大单（> 10%） | VWAP分批 | 分10+批，跟随成交量分布 |

### 3.3 流动性检查建议

| 检查项 | 标准 | 建议 |
|--------|------|------|
| 日均成交量 | 订单量 ≤ 10% | 超过则建议拆分 |
| 买卖价差 | < 0.5% | 超过则谨慎 |
| 盘口深度 | 足够承接 | 不足则等待或拆分 |

## 5. 交易前检查清单

系统在生成建议时会检查以下条件：

| 检查项 | 标准 | 状态 |
|--------|------|------|
| 流动性 | 订单量 ≤ 日均成交10% | ✅ / ⚠️ |
| 价差 | 买卖价差 < 0.5% | ✅ / ⚠️ |
| 停牌 | 未停牌 | ✅ / ❌ |
| 涨跌停 | 未涨停/跌停 | ✅ / ❌ |
| 资金 | 余额充足 | ✅ / ❌ |

## 6. 建议采纳说明

**重要提示**：

1. 以上成本估算和执行建议仅供参考
2. 实际执行成本可能因市场变化而不同
3. 用户应根据自身情况做出最终决策
4. 系统不执行任何自动交易操作
