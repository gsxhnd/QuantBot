# 财报分析系统

## 1. 系统概述

财报分析系统为QuantBot提供基本面分析能力，通过获取、评估财务报表数据，生成财报因子并集成到投资建议中。

**核心能力**：
- 多市场财报数据获取
- 财报质量评估与异常检测
- 财报因子生成与信号融合

**系统定位**：提供财报分析建议，辅助用户决策，不执行自动交易。

## 2. 数据获取

### 2.1 数据源

| 市场 | 数据源 | 数据类型 | 更新频率 |
|------|--------|---------|---------|
| **美股** | yfinance | 三大财务报表、关键指标 | 季报/年报发布时 |
| **A股** | Qlib + akshare | 财务报表、财务指标 | 季报/年报发布时 |

### 2.2 数据类型

**三大财务报表**：
- 资产负债表（Balance Sheet）
- 利润表（Income Statement）
- 现金流量表（Cash Flow Statement）

**关键财务指标**：
- 估值指标：PE、PB、PS、PEG
- 盈利能力：ROE、ROA、毛利率、净利率
- 偿债能力：资产负债率、流动比率、速动比率
- 成长能力：营收增长率、净利润增长率
- 现金流指标：经营现金流、自由现金流

### 2.3 更新机制

| 事件 | 触发动作 |
|------|---------|
| 季报发布 | 自动获取最新财报数据 |
| 年报发布 | 自动获取并更新年度数据 |
| 财报修正 | 标记修正记录，重新评估 |
| 定期检查 | 每日检查是否有新财报发布 |

## 4. 财报质量评估

### 4.1 评估维度

| 维度 | 评估指标 | 权重 | 健康标准 |
|------|---------|------|---------|
| **盈利质量** | 经营现金流/净利润 | 30% | > 1.0 为优，< 0.5 预警 |
| **资产负债质量** | 资产负债率、流动比率 | 25% | 负债率 < 60%，流动比率 > 1.5 |
| **现金流质量** | 自由现金流稳定性 | 25% | 连续正值，稳定增长 |
| **增长一致性** | 收入/利润增长趋势 | 20% | 稳定增长，无异常波动 |

### 4.2 评分计算

```python
def calculate_quality_score(financial_data):
    scores = {
        'earnings_quality': evaluate_earnings_quality(financial_data),
        'balance_sheet_quality': evaluate_balance_sheet(financial_data),
        'cashflow_quality': evaluate_cashflow(financial_data),
        'growth_quality': evaluate_growth(financial_data)
    }
    
    weights = {
        'earnings_quality': 0.30,
        'balance_sheet_quality': 0.25,
        'cashflow_quality': 0.25,
        'growth_quality': 0.20
    }
    
    overall_score = sum(scores[k] * weights[k] for k in scores)
    return overall_score, scores
```

### 4.3 质量等级划分

| 等级 | 分数范围 | 说明 |
|------|---------|------|
| **优秀** | 80-100 | 财务健康，可持续性强 |
| **良好** | 60-79 | 财务状况正常 |
| **一般** | 40-59 | 存在一定风险 |
| **较差** | 20-39 | 风险较高，需谨慎 |
| **危险** | 0-19 | 财务状况堪忧 |

## 5. 异常检测

### 5.1 检测方法

| 检测类型 | 检测方法 | 触发条件 |
|---------|---------|---------|
| **Benford定律检测** | 首位数字分布分析 | 分布偏离 > 阈值 |
| **同比异常波动** | Z-score检测 | \|Z\| > 3 |
| **环比异常波动** | 环比变化率 | 变化 > 历史均值3倍 |
| **行业对比异常** | 行业分位数 | 处于极端分位（<5%或>95%） |
| **逻辑一致性** | 会计恒等式检查 | 资产 ≠ 负债 + 权益 |
| **指标矛盾** | 关联指标对比 | 如：利润增长但现金流恶化 |

### 5.2 Benford定律检测

用于检测财务数据可能存在的造假嫌疑：

```python
def benford_test(numbers):
    """
    检测数字首位是否符合Benford分布
    期望分布：1(30.1%), 2(17.6%), 3(12.5%), ...
    """
    expected = {1: 0.301, 2: 0.176, 3: 0.125, 4: 0.097, 
                5: 0.079, 6: 0.067, 7: 0.058, 8: 0.051, 9: 0.046}
    
    actual = calculate_first_digit_distribution(numbers)
    
    # 卡方检验
    chi_square = sum((actual[d] - expected[d])**2 / expected[d] 
                     for d in range(1, 10))
    
    return chi_square  # 值越大，偏离越严重
```

## 6. 财报因子生成

### 6.1 因子定义

| 因子名称 | 计算方法 | 取值范围 | 说明 |
|---------|---------|---------|------|
| **quality_factor** | 质量评分/100 | 0-1 | 财报质量因子 |
| **value_factor** | 行业PE分位数 | 0-1 | 估值因子（低估值高因子值） |
| **growth_factor** | 增长评分/100 | 0-1 | 增长因子 |
| **risk_factor** | 1 - 风险等级系数 | 0-1 | 风险因子（高风险低因子值） |

### 6.2 因子计算

```python
def generate_financial_factors(stock_code, report_date):
```

## 7. 投资建议集成

### 7.1 信号融合逻辑

**财报因子与价格信号融合**：

```python
def integrate_financial_signals(price_signal, financial_factors):
    # 财报质量调整
    quality_adjustment = financial_factors['quality_factor'] - 0.5
    
    # 风险调整
    risk_penalty = (1 - financial_factors['risk_factor']) * 0.3
    
    # 估值调整
    value_adjustment = (financial_factors['value_factor'] - 0.5) * 0.2
    
    # 综合调整
    adjusted_signal = price_signal * (1 + quality_adjustment * 0.2 
                                       - risk_penalty 
                                       + value_adjustment)
    
    return adjusted_signal
```

### 7.2 置信度调整

| 财报状况 | 置信度调整 |
|---------|-----------|
| 质量评分 > 80 | +10% |
| 质量评分 60-80 | 不调整 |
| 质量评分 40-60 | -10% |

## 8. 风险预警

### 8.1 预警类型

| 预警类型 | 触发条件 | 级别 | 建议动作 |
|---------|---------|------|---------|
| **财报质量恶化** | 质量评分下降 > 20分 | WARNING | 提醒用户关注 |
| **财务异常检测** | 检测到高风险异常 | ERROR | 建议谨慎 |
| **连续亏损** | 连续3季净利润为负 | WARNING | 评估持仓 |
| **现金流恶化** | 经营现金流连续为负 | WARNING | 关注流动性 |
| **债务风险** | 资产负债率 > 80% | ERROR | 高风险预警 |
