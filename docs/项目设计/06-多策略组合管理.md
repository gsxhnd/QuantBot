# 多策略组合管理

## 1. 策略层级结构

```
第一层：品种类（股票/期货/期权）
  ↓
第二层：风格类（价值/成长/动量/均值回归）
  ↓
第三层：具体策略（GRU预测/技术指标/新闻情绪/因子选股）
```

## 2. 权重分配方案

### 2.1 动态权重（推荐）

**配置**：
- 更新频率：每日开盘前
- 回看周期：近20个交易日
- 最小权重：10%
- 最大权重：40%
- 调整阈值：变化 > 5%

**计算逻辑**：基于夏普比率加权 → 应用约束 → 归一化

### 2.2 风险平价

**目标**：每个策略贡献相同风险

**公式**：权重 ∝ 1 / 策略波动率

## 3. 绩效评估

### 3.1 评估指标

| 指标 | 计算方法 |
|------|---------|
| 夏普比 | (收益率 - 无风险利率) / 波动率 |
| 最大回撤 | (峰值 - 谷底) / 峰值 |
| 胜率 | 盈利交易数 / 总交易数 |
| 盈利因子 | 总盈利 / 总亏损 |

### 3.2 动态调整

- **周频评估**：计算各策略夏普比、回撤
- **月频调整**：根据评估调整权重
- **剔除机制**：连续3月夏普 < 0.5 暂停
- **重启机制**：表现恢复后自动恢复权重

## 4. 相关性管理

**目标**：策略间相关性 < 0.5

**监控**：
- 计算策略收益相关矩阵
- 识别高相关策略对
- 降低高相关策略权重
