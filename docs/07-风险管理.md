# 风险管理

## 1. 风险管理框架

**三层建议**：账户级（总资金、总杠杆）→ 策略级（单策略预算）→ 持仓级（单股限制）

**核心理念**：事前建议、事中监控、事后分析

**重要说明**：系统提供风险建议和告警，不执行自动交易。用户根据建议手动决策。

## 2. 核心风险指标

| 指标 | 定义 | 目标阈值 | 监控频率 |
|------|------|--------|--------|
| VaR (95%) | 95%置信度最大亏损 | < 账户2% | 每分钟 |
| 最大回撤 | 峰值到谷底最大跌幅 | < 20% | 每日 |
| 夏普比率 | 风险调整收益 | > 1.0 | 每日 |
| 卡玛比率 | 年收益/最大回撤 | > 0.5 | 每日 |
| 头寸集中度 | 单股票敞口 | < 5% | 每笔交易 |
| 总杠杆率 | 全部头寸/账户资产 | < 2.0 | 每笔交易 |
| 实时回撤 | 当前值与当日高点差 | < 日均150% | 每分钟 |

**VaR计算方法**：历史模拟法、参数法（正态分布）、蒙特卡洛模拟

## 3. 仓位建议

### 3.1 Kelly准则

**公式**：f* = (p × b - q) / b
- p = 胜率，q = 败率，b = 盈亏比
- 实际使用Kelly的1/4或1/2（保守）

### 3.2 仓位建议输出

```json
{
  "position_suggestion": {
    "stock_code": "AAPL",
    "account_value": 100000,
    "suggested_amount": 5000,
    "suggested_percent": 0.05,
    "reason": "Kelly计算建议3%，保守调整为5%",
    "constraints": {
      "single_stock_limit": 0.05,
      "current_position": 0,
      "available": true
    }
  }
}
```

### 3.3 仓位约束建议

| 约束类型 | 建议 |
|---------|------|
| 单股票 | ≤ 账户5%，高波动股票降低 |
| 单行业 | ≤ 账户20% |
| 总杠杆 | ≤ 2.0，高波动期降低 |

## 4. 止损止盈建议与告警

### 4.1 止损建议

| 策略类型 | 建议方式 |
|---------|---------|
| 固定止损 | 建议单笔最大亏损2% |
| 移动止损 | 盈利后建议上移保护利润 |
| 时间止损 | 建议持仓超N天未盈利考虑平仓 |

### 4.2 止盈建议

| 策略类型 | 建议方式 |
|---------|---------|
| 目标止盈 | 建议达到预期收益率（5%、10%） |
| 分批止盈 | 建议盈利后分批减仓 |

### 4.3 价格监控告警

**用户可设置监控条件**：

```json
{
  "price_alert": {
    "stock_code": "AAPL",
    "stop_loss_price": 180.00,
    "stop_loss_percent": -0.05,
    "take_profit_price": 200.00,
    "take_profit_percent": 0.10,
    "trailing_stop_percent": 0.03,
    "alert_method": "claw"
  }
}
```

**告警触发**：

| 条件 | 触发动作 |
|------|---------|
| 价格触及止损价 | 紧急通知用户 |
| 价格触及止盈价 | 通知用户 |
| 移动止损触发 | 通知用户 |
| 回撤超阈值 | 风险预警 |

## 5. 风险预警

| 级别 | 触发条件 | 响应动作 |
|------|---------|---------|
| **WARNING** | 夏普比<1.0、单日回撤>3%、连续3天负收益 | 邮件通知、记录日志 |
| **ERROR** | 实时回撤>日均150%、头寸>10%、杠杆>3.0 | 即时推送、仪表板高亮 |
| **CRITICAL** | VaR突破、单日亏损>5%、系统异常 | 紧急推送、建议平仓 |

**应急预案建议**：
- 市场极端波动：建议停止开仓、评估风险、考虑减仓
- 系统故障：切换备用、手动监控
- 数据异常：停止参考系统建议、验证数据质量

## 6. 压力测试

- 历史情景：2008金融危机、2020疫情、2015 A股股灾
- 假设情景：市场暴跌20%、持仓跌停、流动性枯竭
- 蒙特卡洛：模拟10000条路径，评估尾部风险
