## 系统定位

QuantBot 是一个**量化分析系统**，而非自动交易系统：

- **分析引擎**：数据处理、AI预测、信号生成、风险评估
- **投资建议**：提供买入/卖出建议，附带置信度和风险评估
- **用户决策**：用户接收建议后手动执行交易
- **风险预警**：实时监控，触发条件时告警通知用户
