# 监控告警系统

## 1. 监控维度

| 类别 | 监控指标 | 正常范围 | 告警触发 | 级别 |
|------|---------|---------|---------|------|
| **策略** | 今日收益率 | ±2% | 连续 3 天负收益 | WARNING |
| | 实时夏普比 | > 1.0 | < 0.5 | ERROR |
| | 胜率 | > 45% | < 35% | ERROR |
| **风险** | 实时回撤 | < 日均 5% | > 日均 150% | CRITICAL |
| | VaR 突破 | 在阈值内 | 超历史 95% | CRITICAL |
| | 头寸集中度 | < 5% | > 10% | WARNING |
| **交易** | 成交延迟 | < 2 秒 | > 5 秒 | ERROR |
| | 滑点异常 | 基线 | 超预期 500bps | ERROR |
| | 委托失败率 | < 1% | > 2% | WARNING |
| **数据** | 行情延迟 | < 1 秒 | > 2 秒 | WARNING |
| | 数据缺失 | 0% | 任何缺失 | CRITICAL |
| | 异常值占比 | < 0.5% | > 2% | ERROR |
| **系统** | CPU 使用率 | < 70% | > 80% | WARNING |
| | 内存使用率 | < 75% | > 85% | ERROR |
| | 磁盘可用 | > 20% | < 10% | ERROR |

## 2. 告警分级

| 级别 | 响应动作 |
|------|---------|
| **INFO** | 仅日志记录 |
| **WARNING** | 邮件 + 数据库记录 |
| **ERROR** | 微信推送 + 仪表板 + 邮件 |
| **CRITICAL** | 即时推送 + 自动对冲/平仓 + 紧急停止 |

## 3. 异常检测算法

- **时间序列异常**：Isolation Forest + LOF 混合
- **滑动窗口**：连续 5 日性能下降
- **统计异常**：Z-score > 3σ
- **梯度异常**：指标变化速率过快
